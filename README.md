### РАЗВЕРТКА


✅ Клонировать outletavto

```
git clone https://github.com/shamemask/outletavto.git
```

✅ Перейти в папку abcp_parser

```
cd abcp_parser
```

✅ Загрузить сабмодуль uni_parser

```
git submodule update --init -f
```

# virtual env

python -m venv venv

source venv/bin/activate
